###  有趣的故事：狼和羊

狼被狗所咬，伤势很严重，痛苦地躺在地上，不能外出觅食。这时，他看见一只羊，便请求他到附近的小河里为他取一点水来。他还说：“你给我一点水解渴，我就能自己去寻找食物了。”羊回答说：“如果我给你送水喝，那么我就会成为你的食物。”

这故事告诉我们千万别上那些伪善的恶人的当。
